# Spotify-Recommendation-Project
Built a 30K+ Spotify dataset using the Spotify API and Python (Pandas, NumPy, Requests), cleaned and standardized audio features, and analyzed patterns in song popularity. Developed predictive models to identify key drivers like tempo, energy, and danceability, extracting actionable insights for music trend analysis and recommendation systems.

Collaborators: Manyu Ghildiyal, Craig Sirota, Simon T. Rubinstein
